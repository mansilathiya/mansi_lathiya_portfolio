## 🚀 Introduction

This portfolio website serves as a digital resume and a showcase of my mobile development work. It highlights my skills, experiences, and projects to provide visitors with a better understanding of my capabilities as a **Senior Flutter Developer**.

The portfolio is developed using **Flutter for Web**, demonstrating the power of cross-platform development with smooth animations and a fully responsive design.

---

## ✨ Features

- **Home Section** — An introduction to myself, with animated hero text and a quick overview of what I do.
- **About Section** — A detailed overview of my background, experience, and developer journey.
- **Skills Section** — A visual breakdown of my technical skills including Flutter, Dart, Firebase, REST APIs, GetX, Provider, and more.
- **Projects Section** — A curated showcase of 14+ real-world apps I've built, with Play Store / App Store links and tech stack tags.
- **Experience Section** — A timeline of my professional experience across companies like Squillion Technology, Presiss Technologies, and Appgenix Infotech.
- **Contact Section** — A way for visitors to get in touch with me directly.
- **Responsive Design** — Fully responsive across mobile, tablet, and desktop screen sizes.
- **Smooth Scrolling** — Smooth section navigation for a seamless user experience.
- **Dark / Light Theme** — Theme toggle with preference persistence via `shared_preferences`.
- **Animated Background** — Subtle animated visuals built natively with Flutter.

## ⚙️ Installation

To run this project locally, follow these steps:

**1. Clone the repository**
```bash
git clone https://github.com/mansilathiya/mansi_lathiya_portfolio.git
```

**2. Navigate to the project directory**
```bash
cd mansi_lathiya_portfolio
```

**3. Install dependencies**
```bash
flutter pub get
```

**4. Run the app (Web)**
```bash
flutter run -d chrome
```

**5. Build for production**
```bash
flutter build web --web-renderer canvaskit
```
